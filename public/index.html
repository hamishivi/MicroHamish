MicroHamish - A collection of Microservices

Made for FreeCodeCamp


HamishTime - A timestamp microservice

To use, pass in either a string containing a unix timestamp or a natural date (if it can be parsed by javascript's date object, it can be parsed by this). The API will then return the unixtimestamp and natural language form of that date. If non-valid inputs are sent, it will return null for both properties.

To use, go to: /time/[yourtimehere]

Who Are You? - A Request Header Parser Microservice

It will return your IP address, language and computer operating system/type.

To use, go to: /whoareyou

URL Shortener

Pass in a url (including the http part), and you will get a redirect url in response. If you go to that url, you will be redirected to the original url. If the URL is not valid, there there will be an error instead.

To use, go to: /h/[complete url to create a link for]

Ivison Image - An Image Search Abstraction Layer

It returns a set of metadata for ten image search results. Add ?offset={page number} to paginate through the results.

Go to /image/latest to see the last 20 searched strings, and /image/api/[query] to search.

File Seizer - A file size checker

Upload your file below and this site will return a JSON response of its size in bytes.
